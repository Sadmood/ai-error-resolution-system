# 🤖 Error Expander Multilingual

[![VS Code](https://img.shields.io/badge/VS%20Code-Extension-blue.svg)](https://code.visualstudio.com/)
[![Gemini API](https://img.shields.io/badge/AI-Google%20Gemini-orange.svg)](https://ai.google.dev/)

A modern VS Code extension that uses **Google Gemini AI** to explain cryptic error messages and translate them into your preferred language. Perfect for developers of all levels who want to solve bugs faster and understand their code better!

## ✨ Features (What's New!)

* **🖱️ Hover & Quick Fix:** Trigger the AI explanation seamlessly by hovering over the error squiggles, clicking the Lightbulb (💡) icon, or using a keyboard shortcut!
* **🎨 Modern Native UI:** Beautiful, animated Webview panel that adapts perfectly to your VS Code theme (Dark/Light mode), complete with macOS-style window controls and a sleek copy button.
* **🔒 Secure API Configuration:** Easily configure your personal Gemini API key directly in VS Code Settings (No more hardcoded keys!).
* **🌍 Multilingual Support:** Translate explanations into **Thai**, English, Japanese, Chinese, and many more.
* **🎭 AI Personalities (Tone):** Choose how the AI talks to you:
  * 👨‍🏫 **Teacher:** Detailed, kind, and step-by-step.
  * 💻 **Senior Dev:** Short, direct, and technical.
  * 👶 **ELI5:** Explain Like I'm 5 (super simple terms).
* **🧠 Context Awareness:** Analyzes the exact line of code causing the problem to give you highly accurate solutions and ready-to-use code snippets.

## 🚀 How to Use

You have 3 easy ways to ask the AI to explain an error:

1. **The Hover Method (New!):** Hover your mouse over any error line and click the `💡 คลิกที่นี่เพื่ออธิบาย Error นี้ด้วย AI` link at the bottom of the popup.
2. **The Lightbulb Method:** Click on the error line, click the "Lightbulb" 💡 icon (Quick Fix), and select the explanation option.
3. **The Shortcut Method:** Press `Ctrl + Alt + E` (Windows/Linux) or `Cmd + Alt + E` (Mac) while focusing on the error.

## ⚙️ Installation & Setup

Before using the extension, you need to set up your API Key:

1. Get a **FREE** Google Gemini API Key from [Google AI Studio](https://aistudio.google.com/app/apikey).
2. Open VS Code Settings (`Ctrl + ,` or `Cmd + ,`).
3. Search for `Error Expander`.
4. Paste your API Key in the **Error Expander: Api Key** field.
5. *(Optional)* Customize your preferred **Target Language** and AI **Tone** in the settings.
