# Error Expander Multilingual 🤖

A VS Code extension that uses AI (Google Gemini) to explain error messages and translate them into your preferred language. Perfect for developers who want to understand errors faster!

## ✨ Features

* **💡 AI Explanation:** Get clear, simple explanations for cryptic error messages.
* **🌍 Multilingual Support:** Translate explanations into **Thai**, English, Japanese, Chinese, etc.
* **🧠 Code Context Awareness:** The AI analyzes the specific line of code causing the error for accurate solutions.
* **🎨 Beautiful UI:** Results are shown in a clean Side Panel with syntax highlighting and a Copy button.
* **🎭 AI Personality:** Choose your preferred AI tone (Teacher, Senior Dev, ELI5).

## 🚀 How to use

1.  Click on the "Lightbulb" 💡 icon (Quick Fix) when an error occurs.
2.  Select **"💡 อธิบาย Error นี้ด้วย AI"**.
3.  Read the explanation in the side panel!

## ⚙️ Requirements

* An active Internet connection.
* Google Gemini API Key configured in the source code.
